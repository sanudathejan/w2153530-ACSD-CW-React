Overview

This project is a React-based Single Page Application (SPA) developed for the 5COSC026W Advanced Client-Side Web Development module at the University of Westminster. The application simulates an estate agent platform similar to Rightmove and works fully on the client side without a server.

Features

Property search by type, price, bedrooms, date added, and postcode area

Supports multiple search criteria at the same time

Property results displayed with image, description, and price

Individual property pages with image gallery and tabbed content

Add and remove properties from favourites (button and drag & drop)

Favourites list displayed on the search page

Responsive layout for large and small screens

Client-side security measures applied

Unit testing implemented using JEST

Technologies Used

React JS

JavaScript (ES6)

HTML5

CSS3

JEST

GitHub Pages

Data

Property data stored in a local JSON file

Total of 7 properties covering a range of prices, types, bedrooms, dates, and postcode areas
